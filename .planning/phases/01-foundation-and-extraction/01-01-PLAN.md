# Plan 01-01: Stabilize runnable foundation

<objective>
Make the current FastAPI + React scaffold reproducibly runnable in local development:
backend Python dependencies must install and import cleanly, frontend dev requests must
resolve to the backend without hardcoded URLs, and the initial manager shell must build
without avoidable bundle issues.
</objective>
